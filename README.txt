Ops Center - Quick Start
========================

Folder: D:\Projects\ops-center

RECOMMENDED (no Node.js needed)
-------------------------------
1. Double-click: run.bat
   (or launch.bat / 打开界面.bat / 启动运营中心.bat / 一键打开.bat - same thing)
2. Keep the black window open
3. Browser opens: http://127.0.0.1:8765/app.html

If browser does not open, visit that URL manually.

Dev mode (requires Node.js)
---------------------------
1. Double-click: start.bat
2. Opens http://localhost:5173

Tabs: Tasks | Logistics | Production

Data is in-memory only; refresh resets to sample data.
