🧹 C 盘垃圾清理

检测中… 清理 2345 / 360 / 鲁大师等残留，保护 QQ、微信、百度网盘、WPS

本机操作

通过本地运营中心服务器启动，将在新窗口中运行 PowerShell（先预览，确认后再删）。

下载 Windows 便携版

此工具必须在员工自己的 Windows 电脑上运行，云端服务器无法代替清理本机 C 盘。

下载 disk-cleaner-win.zip
  1. 下载 zip，右键 → 属性 → 勾选「解除锁定」（如有）
  2. 完整解压到文件夹（如 D:\disk-cleaner），勿在压缩包内直接运行
  3. 双击解压文件夹里的 run-cleanup.bat 或「一键清理.bat」
  4. 先预览，输入 Y 后再删除；闪退请运行「诊断环境.bat」

会清理什么

  • 2345、360、鲁大师、猎豹 AI、ksoftmgr 等残留目录
  • 可选：卸载金山毒霸 / 联想管家后的 kingsoft、Lenovo 目录

不会触碰

  • Tencent（QQ / 微信 / 微信输入法）
  • 百度网盘相关目录
  • WPS Office、Cursor、飞书、剪映
