FBA 货件 CSV 批量清洗工具

领星 SKU 库 + FBA 货件 CSV → 瀚海万博 B2B 单票导入模版（.xlsx，含服务渠道下拉），批量打包下载。

领星 SKU 库（GitHub 团队共享）

团队数据保存在 GitHub Gist（与运营中心物流/任务同一套云端）。每次打开为空白状态，不会带上一位同事的本机数据；需要 SKU 库时点「从团队库更新」。填完可「同步到团队库」供他人调用。也可从领星 ERP 导出 CSV/XLS 导入；相同 SKU 不覆盖，只新增。产品图可单独导入领星「导出产品-按SKU」ZIP，按 SKU 合并到库中。

1. 上传 FBA 货件文件（支持多选）

上传货件 CSV 后，须为每个货件填写「货件追踪编号」（例：4KKJA95Q，每票不同）；再按 SKU 与领星库自动匹配申报配置。

2. 产品申报配置（领星库自动匹配）

紫色「领星库」来自领星导入；申报单价、材质等须手填后点「保存到 SKU 库」入库，下次同 SKU 自动匹配（绿色「已入库」）。

已配好 待补全 模版必填空缺 FBA 已带入 领星库已匹配

3. 导出设置

与瀚海官方模版一致，导出 Excel 对应单元格带下拉：服务 B2、报关 F6、交税 F8、币种 F17。美国渠道默认「美元」；欧洲「欧元」；英国「英镑」。非单独报关选「买单报关」，单独报关选「报关退税」。产品图：先导入领星 ZIP，库统计须显示「含产品图」；导出 XLSX 时 S 列嵌入缩略图。
