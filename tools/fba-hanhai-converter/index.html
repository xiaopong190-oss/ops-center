领星 SKU 库 + FBA 货件 CSV → 瀚海万博 B2B 单票导入模版（.xls），批量打包下载。

领星 SKU 库（GitHub 团队共享）

数据保存在 GitHub Gist（与运营中心物流/任务同一套云端），全员可「从团队库更新」调用。也可从领星 ERP 导出 CSV/XLS 导入；相同 SKU 不覆盖，只新增。只读备份：docs/data/shared-lingxing-sku-db.json。

1. 上传 FBA 货件文件（支持多选）

上传货件 CSV 后，将按 SKU 与领星库自动匹配，并显示本批产品申报配置。一份 CSV 可含多个 SKU。

2. 产品申报配置（领星库自动匹配）

紫色「领星库」来自领星导入；申报单价、材质等须手填后点「保存到 SKU 库」入库，下次同 SKU 自动匹配（绿色「已入库」）。
