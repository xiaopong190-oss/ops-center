泓森拓创科技 - 使用说明
========================

项目位置：D:\Projects\ops-center

【推荐】方式一：双击启动（无需安装 Node）
----------------------------------------
1. 双击「打开界面.bat」或「启动运营中心.bat」
2. 浏览器会自动打开 http://127.0.0.1:8765
3. 不要关闭弹出的黑色命令行窗口

若上面失败，双击「一键打开.html」（需联网）

方式二：开发模式（需安装 Node.js）
----------------------------------
1. 双击「start.bat」
2. 首次会自动 npm install
3. 浏览器打开 http://localhost:5173

功能说明
--------
- 任务跟进：任务卡片、节点进度、卡点
- 物流头程：在途统计、搜索、运输筛选、到港提醒
- 精品生产：生产阶段、异常记录
- 工具：FBA 计算器、分仓、推广追踪、C 盘垃圾清理（本机工具）

C 盘清理（本机）
----------------
- 工具页 → C 盘垃圾清理 → 本机模式下可扫描/预览/清理
- 云端部署时员工下载 packages/disk-cleaner-win.zip 在本机运行
- 打包 zip：powershell -File deploy\pack-disk-cleaner.ps1

数据保存在浏览器内存中，刷新页面会重置为示例数据。
